python基础

输入

1.跳过空格：
a,b=map(int,input().split())

2.字符/字符串输入：
a=input()—永远输入的是字符串

输出

转换（大小写，字符，ASCII）

  • 转大写
    s=input()
    print(s.upper())

  • 转小写
    s=input()
    print(s.lower())

  • 首字母转大写
    s=input()
    print(s.capitalize())

  • 字符->ASCII
    print(ord(‘A’))

  • ASCII->字符
    print(chr(65))

注：

判断类型

  • 是否为数字
    print(a.issdigit())

  • 是否为字母
    print(a.isalpha())

  • 是否为字母或数字
    print(a.isalnum)

  • 是否为小写
    print(a,islower())

  • 是否为大写
    print(a.isupper())

切片

例题：
输入一个123.4，输出4.321

a=input()
print(a[::-1])

例子：

a=”abcdef”

  • a[1:4]
    表示：从下标1开始，至下标4结束，不包含4
    输出 “bcd”

  • a[::2]
    表示：从头开始，到尾结束，每次跳两格
    输出 “ace”

  • a[::-1]
    表示：start=空；end=空，step=-1
    从最后开始，向前走，每次走1格，直到最前面
    输出：”fedcba”

  • a[-1]
    表示：查看最后一个字符

  • 判断回文串
    s=input()

if s==s[::-1]:
print(“yes”)

else:
print(“no”)

保留小数

保留两位小数

1.round(num，2)

2.print(f”{num1:.3f}”)

向下取整

import math
print(math.floor(3.9))

输出：3

2.int()
对于正数来说:
print(int(3.9))

注意：floor=往左边最近的整数

所以对于负数来说：
print(int(-3.9))
—>-3

print(math.floor(-3.9))
—>-4

向上取整

import math
math.ceil(3.1)

输出：4

补前导零

eg：
h=7；m=8
要求输出：07:08

print(f”{h:02d}”)

注：宽度至少两位
1.不足两位，前面补一位；
2.超过两位，不管
